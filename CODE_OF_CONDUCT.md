# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1.

In summary: be respectful, assume good faith, focus on what's best for the project, and treat fellow contributors with kindness. Harassment, personal attacks, and discriminatory language are not tolerated.

To report conduct issues, contact the maintainers directly. Reports will be reviewed and addressed promptly and confidentially.
